Функции, основы

Переменные, объявленные внутри функции, видны только внутри этой функции. Внешняя переменная используется, только если внутри функции нет такой локальной. Функция обладает полным доступом к внешним переменным и может изменять их значение. 

      // объявление функции
      function имя(параметры, через, запятую) {
        ...тело...
      }

      // вызов функции
      имя(аргументы, через, запятую);

Значения по умолчанию

Если мы хотим задать параметру значение по умолчанию, мы должны указать его после =

По умолчанию может быть задан как примитив (строка, число,...), так и более сложное выражение, которое бы вычислялось и присваивалось при отсутствии параметра. Например: 

      function showMessage(from, text = anotherFunction()) {
        // anotherFunction() выполнится только если не передан text
        // результатом будет значение text
      }

Результат функции с пустым return или без него – undefined

Выбор имени функции

Функция – это действие. Поэтому имя функции обычно является глаголом. Как правило, используются глагольные префиксы, обозначающие общий характер действия, после которых следует уточнение.

Функции, начинающиеся с…
  • "show" – что-то показывают,
  • "get…" – возвращают значение,
  • "calc…" – что-то вычисляют,
  • "create…" – что-то создают,
  • "check…" – что-то проверяют и возвращают логическое значение, и т.д.

Одна функция – одно действие Функция должна делать только то, что явно подразумевается её названием. И это должно быть одним действием. Два независимых действия обычно подразумевают две функции, даже если предполагается, что они будут вызываться вместе (в этом случае мы можем создать третью функцию, которая будет их вызывать).
